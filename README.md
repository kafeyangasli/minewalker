*Minesweeper but make it Wumpus-themed*

## Introduction

Minewalker merupakan game tile-based yang terinspirasi dari Minesweeper, tetapi dibuat lebih berorientasi pada pergerakan player. Player memilih titik spawn pada map berukuran $M \times N$, lalu harus membuka seluruh tile aman tanpa menginjak tile bom.

Game ini dibangun menggunakan Java Swing dengan tampilan sprite sederhana, kontrol keyboard penuh, sistem flag, countdown timer opsional, audio manager, dan penyimpanan konfigurasi/performa permainan.

> Projek ini disusun dalam rangka memenuhi syarat penugasan mata kuliah **Pemrograman Berorientasi Objek**

## Features

- Kustomisasi ukuran map dan persentase tile bom
- Pemilihan titik spawn sebelum map digenerasi
- Flagging untuk mencegah player tidak sengaja masuk ke tile berbahaya
- Countdown timer opsional dengan batas waktu 1 sampai 5 menit
- Soundtrack dan SFX yang unik
- Persistensi data melalui storage untuk menyimpan konfigurasi dan hasil permainan

## Controls

- Menus: `W/S` atau `Up/Down` untuk memilih menu, `Enter` atau `Space` untuk konfirmasi.
- Gameplay Configuration: `W/S` atau `Up/Down` untuk memilih , `A/D` atau `Left/Right` untuk mengatur value.
- Spawn: `WASD` atau tombol arah pada keyboard untuk menentukan titik spawn, lalu `Enter` atau `Space`.
- Gameplay: `WASD` atau tombol arah untuk bergerak.
- Flag: tekan `F`, lalu `WASD` atau tombol arah untuk menaruh flag.
- Pause: `Esc`, lalu `W/S` atau `Up/Down`, dan `Enter` atau `Space` untuk konfirmasi.
- Gameplay Finished: `Space` untuk memulai kembali, `Esc` untuk kembali ke menu.
